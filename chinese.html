游戏设定：

  1. 世界观

  1. 宇宙的神明创造了一个擂台，名为继承擂台。冕之擂台会将宇宙各地的强者传送至此进行决斗，传说最后的赢家将会得到神的继承。
  2. 宇宙之大，他孕育了千万人类种族，每个种族都有不同的功法，却无万众之王。
  3. 神的力量正在衰弱，他正在寻找着合适的人来夺舍。
  4. 他是上古古神战争残留之后的一丝邪念，吸收了其他古神的遗骸成神。
  5. 上古古神们的残害幻化了生机，说以现在每个种族都是一位古神的后裔。
  6. 每一位古神的后裔都有继承祖先古神力量的潜力

  3. 积分的获得取决于每局游戏的表现。

5V5团队战：

赛制：

  1. 采用一号地图。
  2. 采用同段位匹配机制。
  3. 双方五人同时上场进行5V5。
  4. 每位玩家选择一个角色，并每局游戏一个角色只能被一个玩家使用。
  5. 每个玩家有一次复活机会，死去30秒之后才能复活。
  6. 如果一方玩家全部全在死亡中，则游戏结束，判此方为输，敌方为赢。
  7. 如果十分钟内没有胜负，人数多的一方赢，如果人数一样，伤害多的一方的人赢。
  8. 如果敌方剩下三人或以上，并血量都在一般以上，我方剩下一个人，触发彩蛋 ”英雄还是狗熊”。
  1. “英雄还是狗熊” :我方唯一在场角色, 重设所有技能冷却, 恢复生命到满。

积分：

  • 根据玩家选择职业进行排列,
  • 如果赢，根据排名加分
  • 得分排名第一，加6分
  • 得分排名第二，加5分
  • 得分排名第三，加4分
  • 得分排名第四，加3分
  • 得分排名第五，加3分

8人单人组淘汰赛✨：

赛制：

  • 采用一号地图。
  • 地图上就会有两名玩家进行1V1。
  • 四场比赛同时进行
  • 十个队伍每个由一个人组成。
  • 采用同段位匹配机制。
  • 每位玩家选择一个角色，并每局游戏一个角色只能被一个玩家使用。
  • 如果一方玩家死亡，则游戏结束，判此方为输，敌方为应并继续挑战剩余玩家。
  • 每次战斗开始，玩家角色状态恢复到初始。

  积分：

  • 第一场输了扣10分
  • 第二场输了扣5分
  • 第三场输了分数不加不减
  • 第四场输了加3分
  • 第五场输了加4分
  • 冠军获得10分奖励。

  • 如果一方玩家全部全在死亡中，则游戏结束，判为输。

Shader "Custom/AutoToonShader"

{

    Properties

    {

        _MainTex ("Texture", 2D) = "white" {}

        _OutlineColor ("Outline Color", Color) = (0,0,0,1)

        _OutlineWidth ("Outline Width", Range(0.001, 0.03)) = 0.005

    }

    SubShader

    {

        Tags { "RenderType"="Opaque" }

        LOD 200

       

        CGPROGRAM

        #pragma surface surf Lambert

       

        struct Input

        {

            float2 uv_MainTex;

        };

       

        sampler2D _MainTex;

        fixed4 _OutlineColor;

        float _OutlineWidth;

       

        void surf (Input IN, inout SurfaceOutput o)

        {

            fixed4 c = tex2D(_MainTex, IN.uv_MainTex);

            o.Albedo = c.rgb;

            o.Alpha = c.a;

            // 计算环境光照

            fixed3 ambient = UNITY_LIGHTMODEL_AMBIENT.xyz;

            // 计算漫反射光照

            fixed3 lightDir = normalize(_WorldSpaceLightPos0.xyz);

            float diff = max(0, dot(o.Normal, lightDir));

            fixed3 diffuse = diff * _LightColor0.rgb;

            // 自动调整材质颜色，高光和阴影

            fixed3 lightColor = ambient + diffuse;

            if (lightColor.r < 0.5)

                o.Albedo *= 0.5;

            else if (lightColor.r < 1.0)

                o.Albedo *= 0.8;

            // 添加描边效果

            outline(IN.uv_MainTex, c);

        }

        void outline(float2 uv, inout fixed4 color)

        {

            float2 d = fwidth(uv);

            float a = tex2D(_MainTex, uv).a;

            float t = smoothstep(0.5 - d, 0.5 + d, a * (1 - a));

            color.rgb = lerp(color.rgb, _OutlineColor.rgb, t);

        }

        ENDCG

    }

    FallBack "Diffuse"

}
